# 🔥 BIG NIK BURGERS - MENÚ ÉPICO

**El mejor menú del mundo para tu marca de burgers y pollo crispy.**

- ✨ Diseño cinematográfico con animaciones fluidas (Framer Motion)
- 🎨 Composición artística de nivel mundial
- 📱 Totalmente responsivo (móvil y PC)
- 🔥 Burgers mejorados con IA (food porn style)
- 🍗 3 sándwiches de pollo hiperrealistas
- 👨‍🍳 Presentación épica del Chef Bignik
- ⚡ Optimizado para rendimiento
- 🌐 Listo para GitHub Pages

## 📦 Contenido

- **index.html** - Página principal compilada
- **assets/** - Logos, imágenes y recursos
- **public/** - Fotos de burgers, pollo y chef

## 🚀 Cómo usar en GitHub Pages

1. Crea un repositorio en GitHub llamado `bignik-menu`
2. Sube todo el contenido de esta carpeta
3. Ve a Settings → Pages → Source: Deploy from a branch → main
4. Tu sitio estará en: `https://tu-usuario.github.io/bignik-menu`

## 🎯 Estructura de Páginas

1. **Home** - Bienvenida épica
2. **Burgers** - Smash, Double Smash, Crispy Onion
3. **Chicken** - Crispy, Spicy, BBQ
4. **Chef** - Presentación del Chef Bignik

## 🎮 Navegación

- **Flechas del teclado** - Navegar entre páginas
- **Puntos laterales** - Click para ir a página específica
- **Botones** - Navegación directa
